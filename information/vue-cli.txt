vue-cli使用步骤
1. npm install vue-cli -g
2. vue init webpack 项目名字
3. 在安装过程中可以一直回车到vue-router
4. 终端上会有一个提示 cd 项目名字
5. 进入项目中 执行 npm run dev
6. 在浏览器中输入 localhost:8080
